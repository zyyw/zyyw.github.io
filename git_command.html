Check And Create a branch: git branch: list all branches on local machine git branch new_branch_name: create a new branch Switch Between Branches: git checkout dest_branch_name: switch to destination branch Delete One Branch: git branch -d branch_to_delete: delete a branch Rename One Branch: git branch -m branch_old_name branch_new_name: move a branch from its old name to a new descriptive name
+++ point to the same commit, but points to different branches
