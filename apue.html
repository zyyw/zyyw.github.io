Installation
step1. go official website, download the whole source code package.
step2. untar it
step3. go to $Dir/apue.3e/
step4. type command "make" in terminal
step5. sudo cp $Dir/apue.3e/include/apue.h     /usr/include/
step6. sudo cp $Dir/apue.3e/lib/libapue.a     /usr/lib/
step7. test. Typing "gcc source_code.c -lapue" in terminal

Threads and Thread IDs
All Threads within a process share the same address space, file descriptors, stacks, and process-related attributes.
