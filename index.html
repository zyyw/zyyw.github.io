Check And Create a branch:
  • git branch: list all branches on local machine

  • git branch new_branch_name: create a new branch


Switch Between Branches:
  • git checkout dest_branch_name: switch to destination branch


Delete One Branch:
  • git branch -d branch_to_delete: delete a branch


Rename One Branch:
  • git branch -m branch_old_name branch_new_name: move a branch from its old name to a new descriptive name


Remote Branch:
  • git remote add origin URL: add a remote repo to local origin alias

  • git remote rm origin: remove the origin track of remote repos

  • git branch -r: display remote branches only

  • git branch -a: display both remote and local branches


Clone a Remote Repos:
  • git clone URL

  • git clone URL directory_new_name
