Check And Create a branch:
git branch: list all branches on local machineg
git branch new_branch_name: create a new branchg

Switch Between Branches:
git checkout dest_branch_name: switch to destination branchg

Delete One Branch:
git branch -d branch_to_delete: delete a branchg

Rename One Branch:
git branch -m branch_old_name branch_new_name: move a branch from its old name to a new descriptive nameg

Remote Branch:
git remote add origin URL: add a remote repo to local origin aliasg
git remote rm origin: remove the origin track of remote reposg
git branch -r: display remote branches onlyg
git branch -a: display both remote and local branchesg

Clone a Remote Repos:
git clone URLg
git clone URL directory_new_nameg
